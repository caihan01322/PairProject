**项目介绍**
本项目是一个收集了部分论文的平台，可以供用户搜索、查看论文和查看频率最高的关键词


* 首页
    - 展示论文列表
    - 可以进行论文搜索(支持同时搜索关键词、题目)
* 论文管理
    - 可以搜索论文再对论文进行删除
* 论文分析
    - 展示三大顶会的热词走势
    - 展示关键词图谱
